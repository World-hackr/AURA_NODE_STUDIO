# AURA Report Archive

This folder is archive/reference material, not the active implementation center of `AURA Node Studio`.

What lives here:

- conference draft material
- synopsis and report exports
- presentation exports
- weekly progress reports
- generated diagrams used in academic/report work

Important notes:

- several files are different export formats of similar content
- `ppt_assets/conference_media/` is the kept image set after a duplicate `report_media/` copy was removed
- hidden `~$` files are temporary Office lock files and are not meaningful project assets

Use this folder for background, narrative framing, and prior formal writeups.
Do not treat it as the source of truth for the active firmware/tooling structure.
