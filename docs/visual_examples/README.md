# Visual Examples

This folder is the visual-design playground for `AURA Node Studio`.

Use it to keep UI and system thinking visible instead of burying everything in long markdown files.

## What To Use

- `drawio/` for cleaner structured diagrams and screen maps
- `mindmap/` for hierarchy-first thinking such as menu trees and grouped ideas

## Suggested Rule

Use each tool for a different job:

- Draw.io: final-ish structure
- Mindmap: high-level tree thinking

Do not try to force one tool to do everything.

## Suggested Naming

- `host_menu_tree.*`
- `host_setup_flow.*`
- `node_mapping.*`
- `radio_confirmation.*`

## How To Open

- `.drawio` files: open normally in VS Code with Draw.io Integration
- `.km` files: open with your mindmap extension
