# Shared Contracts V1

This folder contains machine-readable contract artifacts for the first hardware-first AURA slice.

Current schemas:

- `inventory_item.schema.json`
- `storage_location.schema.json`
- `locate_target.schema.json`
- `locate_session.schema.json`
- `part_lookup_result.schema.json`
- `contract_index.json`

These schemas mirror the planning definitions in `docs/DATA_CONTRACTS_V1.md`.

They are intended to keep `AURA Host`, `AURA Phone`, and supporting studio/tooling surfaces aligned on the same stable vocabulary.
