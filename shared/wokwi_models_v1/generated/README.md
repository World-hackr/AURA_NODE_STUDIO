# Generated Wokwi Extracts

This folder holds machine-generated JSON extracts from the local offline Wokwi source tree.

These files are generated by:

- `local_tools/component_import/scripts/extract_wokwi_source_models.mjs`

## Purpose

Use this folder to:

- inspect what Wokwi exposes directly
- compare vendor pin data against AURA snap-point behavior
- bootstrap broader Wokwi normalization work without manually curating every part first

## Rule

These generated extracts are not the final curated AURA Wokwi model files.

They are raw parsed source summaries.

The curated bridge files that `AURA Studio` currently uses still live under:

- `shared/wokwi_models_v1/models/`
